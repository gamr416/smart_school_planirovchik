"""Документация для Telegram-бота "Распорядок Дня""""

Описание:

Бот "Распорядок Дня" — это инструмент, предназначенный для помощи пользователям в составлении ежедневного расписания. С помощью бота пользователи могут добавлять различные дела и события, получать уведомления, помечать задачи как важные и отслеживать статистику выполненных задач.

Основные функции

1. Добавление дел

Пользователь может добавить любое количество дел, указав название, время окончания(дедлайн), а также приоритетность задачи.

Пример команды:
/Добавить --> Учёба (Ваша категория) --> 13:00 16.08.2025 --> Высокий приоритет

2. Просмотр списка дел

Для просмотра всех запланированных дел на день или неделю, пользователь может использовать команду /Показать.

Пример команды:
/Показать

3. Удаление дел

Чтобы удалить ненужную задачу, достаточно указать её айди.

Пример команды:
/Удалить 12345678

4. Голосовой ввод

Пользователи могут отправить голосовое сообщение боту, если не хотят писать в ручную. Бот распознает речь и преобразует в текст.
